# Persönliches Webseiten-Projekt
Hier folgt ein persönliches Webseiten-Projekt zur Präsentation meiner Projekte, Kenntnisse sowie Fähigkeiten im Umgang mit künstlicher Intelligenz (Claude.ai, Claude Code, Claude Design) unter Anwendung von React und Material UI.

Aktuell befinde ich mich noch im Rahmen einer Umschulung in "Ausbildung" zum Fachinformatiker mit Schwerpunkt Anwendungsentwicklung und versuche meine Kenntnisse im Bereich Webentwicklung, nativer Android Appentwicklung, Web Progressive Apps sowie der Spieleentwicklung, jeweils im Frontend als auch Backend, UI-/UX, Security- und Architektur- Bereich zu festigen. 

Der administrative Bereich rund um Netzwerktechnik, IoT und Optimierung sowie der Entwicklung geeigneter Strukturen soll mein Wissensspektrum breit absichern um möglichst optimal in der Entwicklung spezifischer Softwareprodukte agieren zu können.
